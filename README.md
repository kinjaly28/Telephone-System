## 📞 Telephone Contact System (C - DSA Project)

This project is a **menu-driven Telephone Contact Management System** implemented in **C** using a **Doubly Linked List**. It allows users to efficiently store and manage contacts with dynamic memory allocation.

### 🚀 Features

* Add new contacts
* Display all contacts
* Search contact by name
* Delete contact
* Update contact details
* Navigate contacts (forward & backward)

### 🛠️ Technologies Used

* C Programming
* Doubly Linked List
* Dynamic Memory Allocation

### ⚙️ How It Works

Each contact is stored as a node containing:

* Name
* Phone Number
* Previous & Next pointers

The system uses a menu-driven interface for easy interaction and performs CRUD operations efficiently.

### 🎯 Learning Outcomes

* Understanding of linked lists
* Memory management in C
* Implementation of real-world applications using DSA

### ▶️ How to Run

1. Compile the code:

   ```bash
   gcc filename.c -o output
   ```
2. Run the program:

   ```bash
   ./output
   ```

### 📌 Conclusion

This project demonstrates how **Doubly Linked Lists** can be used to build a flexible and efficient contact management system compared to static data structures.
